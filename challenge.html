WorldPose Challenge 2025

WorldPose is a novel dataset for advancing research in multi-person global pose estimation from monocular videos, featuring footage from the 2022 FIFA World Cup.

Challenge Overview

We are organizing a challenge at CVPR 2025 to benchmark the performance of global 3D human pose estimation methods on the WorldPose dataset. The challenge will feature a public leaderboard and a workshop at CVPR 2025. Stay tuned for more information!

Rules

Participants are encouraged, though not required, to train their algorithms using publicly available datasets. Predictions should include tracks of multiple players, with each track providing the body pose, global orientation, and translation for every frame. Submissions must be uploaded and publicly displayed on the leaderboard. Please specify any external data used for training in the 'method description' when uploading results to the evaluation server.

By the challenge deadline, results for both the validation and test sets must be submitted to the evaluation server. Validation results will be made public for performance diagnosis and visualization. Competitors' algorithms will be evaluated based on feedback from human judges, and the top-performing teams will be awarded prizes.

Evaluation

The evaluation server will use the evaluation code provided here, which calculates several common metrics, including PA-MPJPE, G-MPJPE, and Per-Meter Drift, as described in our paper.

To obtain results on the test set, where ground truth annotations are hidden, participants must submit their generated results to the evaluation server. The same evaluation code mentioned above will be used to evaluate the generated captions on the test set.
